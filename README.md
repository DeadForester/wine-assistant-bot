# Wine Assistant Bot

Интеллектуальный Telegram-бот для винного магазина на базе Yandex Cloud Assistant API.  
Позволяет пользователям искать вина по параметрам, получать рекомендации и оформлять заказы.  
Администратор может отслеживать активные заказы через веб-интерфейс.

---

## Возможности

### Для пользователя (в Telegram):
- Поиск вин по названию, стране, цвету, кислотности
- Сортировка по цене (дороже/дешевле)
- Добавление вин в корзину
- Просмотр содержимого корзины
- Передача диалога оператору

### Для администратора (веб-интерфейс на `:8080`):
- Список всех активных заказов (сессий с непустой корзиной)
- Просмотр корзины по `session_id`
- История переписки с пользователем
- Автообновление каждые 10 секунд
